# Sales & Profit Analysis using SQL

## Project Overview
This project analyzes sales and profit data using MySQL queries, views, functions, stored procedures, CTEs, and window functions.

The project focuses on:
- Sales analysis
- Profit analysis
- Customer insights
- Fiscal year & fiscal quarter analysis
- Region-wise performance
- Category-wise analysis
